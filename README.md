# Greenhouse AR (iOS Only Version)

This version is optimized for iPhone and iPad using Apple AR Quick Look.

## How to Use

1. Replace:
   assets/greenhouse.usdz

2. Upload all files to a GitHub repository.

3. Enable GitHub Pages:
   Settings → Pages → Deploy from branch → main → /(root)

4. Open the site on Safari (iPhone/iPad) and tap "View in AR".

Must be served over HTTPS (GitHub Pages is fine).
